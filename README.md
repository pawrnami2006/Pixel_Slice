# Pixel Slice 🎮🍉

A hand gesture controlled Fruit Ninja inspired game built using OpenCV, MediaPipe, and custom pixel art assets.

## Features

* Real-time hand tracking using MediaPipe
* Index finger controlled slicing
* Custom pixel art fruits
* Fruit slicing mechanics
* Score system
* Lives and Game Over screen

## Tech Stack

* Python
* OpenCV
* MediaPipe
* NumPy

## Current Progress

✅ Hand tracking completed

✅ Fruit spawning in progress

🚧 Slicing mechanics in progress

## Assets

All fruit and UI assets were created as custom pixel art for this project.
